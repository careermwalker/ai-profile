AI Message Verification Instructions

Hello! This page explains how to verify whether a message sent to Michael Walker was generated by a human or by an AI system following the declared provenance protocol.

1. Check the AI Disclosure Label

All AI-generated messages should include the label:

#AIGENERATED

If the message does not contain this label, it is assumed to be human-authored unless otherwise verified by heuristics.

2. Verify the Signature (Optional but Recommended)

If the sender provides a signed token, you can verify it using the public key hosted here:

  https://raw.githubusercontent.com/careermwalker/ai-profile/main/ai_public.pem

Example using Python (PyNaCl):

from nacl.signing import VerifyKey
import base64

# Load public key (PEM)
with open("ai_public.pem", "rb") as f:
    key_data = f.read()
vk = VerifyKey(base64.b64decode(key_data.split(b'\\n')[1]))

# Verify the signature
message = b"Message text here"
signature_b64 = "provided_signature_here"
signature = base64.b64decode(signature_b64)
vk.verify(message, signature)
print("Signature verified!")

  • AI-generated messages must include #AIGENERATED and a signature URL or token if possible.
  • Use the public key above to verify any signed message.
  • If no label or signature exists, apply heuristic evaluation (optional).
